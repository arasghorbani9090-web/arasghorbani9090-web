I build real products end-to-end — from PostGIS queries and Rust simulation cores to the pixel-level polish of a mobile storefront. I'm currently focused on **Souq**, a map-first marketplace bringing the Gulf's informal commerce — shops, freelancers, and independent sellers — onto a single interactive map.

I care about shipping. Every project below is a working system, not a slide deck.

- 🤝 **Open to:** technical co-founder conversations, accelerator intros, and engineers who like hard product problems.
